About Buddy

  • "Buddy", is an intelligent personal voice assistant application developed in Python Language.

  • Buddy works in both online and offline mode. User and Buddy both will interact with the help of GUI application window. It supports both voice and text commands for performing operations.

  • The main module used in this project, nltk, numpy, gTTS, speech recognition, Pyttsx3, Wikipedia, Pyjoke, and TKinter.
